# PlayerPro Job Scheduler #

## Getting Started ##

### Create virtualenv, activate it, and install dependencies. ###

#### Python 2.7.9 ####

##### Mac #####
    brew install python
    pip install virtualenv

##### Ubuntu #####
    sudo apt-get install python-pip
    sudo pip install virtualenv

##### Virtualenv #####
    virtualenv .
    . bin/activate
    pip install -r requirements.txt

### Run the api ###

    python run_api.py
